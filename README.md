# 🤖 AI Conversational Data Science Tutor

🚀 An interactive AI-powered chatbot that answers Data Science-related questions using Google's **Gemini AI** and **LangChain**.  
This project provides a **WhatsApp-style chat interface** built with **Streamlit**.

---

## 📌 Features
✅ **Conversational AI** – Ask multiple questions, and see past chats like WhatsApp.  
✅ **Google Gemini AI Integration** – Powered by **LangChain** & **Gemini 1.5 Pro**.  
✅ **Interactive UI** – Streamlit-based interface with a clean and user-friendly layout.  
✅ **Persistent Chat History** – Messages remain visible like a real chat application.  
✅ **No Backend Required** – Works fully in Streamlit without FastAPI.
